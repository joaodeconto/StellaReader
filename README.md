# StellaReader

A simple Android PDF and EPUB reader built for my daughter Stella.

The project began with a practical goal: give her a quiet, uncomplicated place to read books, automatically return to the last page, and save meaningful passages without the clutter of a commercial reading app.

## Core features

- PDF and EPUB support
- Automatic reading-position recovery
- Page and passage bookmarks
- Simple local library
- Easy file import and Android sharing
- Distraction-free reading interface
- Offline-first storage
